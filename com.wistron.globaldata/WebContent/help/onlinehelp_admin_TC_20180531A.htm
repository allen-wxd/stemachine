簡介

STEMachine 是一款教授學生體驗並使用 Python 語言程式設計的教學平臺。STEMachine 以機械手臂為媒介，讓學生能夠通過基本的 Python 語言應用，操縱機械手臂，體驗專案開發的經驗與樂趣。

本文將為您詳細介紹如何使用管理員帳號匯入和維護系統上的學生和教師帳號。

聲明：

  • 實際產品中的圖示、文字及實際操作可能與本文檔存在細微差異，請以實際為准。

登入

打開瀏覽器，訪問 STEMachine 網址：

http://ServerIP:8080/htm/login.html

其中，請將 ServerIP 替換成您實際的 STEMachine 伺服器 IP 地址。8080 為默認埠，如果您在系統安裝時設定了其他埠，請替換為其他埠號。如果您不知道伺服器 IP 位址，請諮詢幫您搭建 STEMachine 系統的技術服務人員。

注意：建議您使用 Chrome 瀏覽器訪問 STEMachine。

輸入管理員帳號的使用者名稱和密碼：

使用者名稱：admin

初始密碼：123456

點選“登入”。

您可能想要：

  • 返回到頂部

學生管理

使用管理員帳號登入後，點選左側的“學生管理”，管理學生帳號。您可以對學生帳號做以下操作：

  • 匯入學生
  • 課程匹配
  • 刪除學生
  • 修改學生
  • 匯出學生清單
  • 查找學生

匯入學生

在學生可以使用 STEMachine 系統之前，您首先需要匯入學生帳號。

有以下兩種方式可以添加學生帳號：

  • 批量匯入學生
  • 手動添加學生

建議您使用批量匯入的方式，以節省時間。您只需一次操作，就可以添加所有學生帳號。

批量匯入學生

  1. 點選“學生管理” >> “匯入資料”。
  2. 在“參考範例”一欄，點選“学生.xlsx”，下載學生帳號範本檔。範本檔為 Excel 格式，使用前請確保您已經安裝了 Microsoft Excel。
  3. 用 Excel 打開您下載的範本檔。您將會看到諸如下圖格式的表格：

  4. 在表格中填入每個學生的學籍號、年級、班級、姓名和學號。姓名長度需在 2-8 個字元範圍內。其中，學籍號即為學生的登入帳號，因此，每個學生的學籍號不得重複。如下圖所示（圖中文字僅為舉例說明用，實際操作中請填入真實的學生資訊）。填寫完畢後，請儲存檔案並退出 Excel。

  5. 在 STEMachine 頁面，前往“學生管理” >> “匯入資料”。點選“選擇檔案”，選擇您剛才製作好的 Excel 檔案。
  6. 請耐心等待文件上傳。上傳完畢後，在“選擇檔案”按鈕右側，會出現您所選檔案的檔案名。完成後，點選“匯入資料”。
  7. 匯入成功後，會出現提示，您可以在學生清單中看到匯入的學生帳號。如果沒有看到帳號，可以嘗試刷新頁面。如果匯入失敗，螢幕會出現失敗提示消息。請對比範本，檢查您的 Excel 檔案格式是否正確。

注意：匯入帳號的密碼將自動設為預設密碼“123456”。

手動添加學生

  1. 點選“學生管理” >> “新增”。
  2. 填寫學生資訊。

  3. 點選“儲存”。

您可能想要：

  • 匯入教師
  • 添加課程
  • 修改學生
  • 刪除學生
  • 返回到學生管理
  • 返回到頂部

課程匹配

添加了學生帳號之後，您還可以為學生分配課程。在分配課程之前，請確保您已經匯入了教師帳號並添加了相應的課程。

  1. 點選左側的“學生管理”。
  2. 選中您要分配課程的學生（支持多選）。
  3. 點選上方的“課程匹配”按鈕。
  4. 勾選您要給這些學生分配的課程（支持多選）。
  5. 點選“儲存”按鈕。

您可能想要：

  • 修改學生
  • 匯入學生
  • 返回到學生管理
  • 返回到頂部

刪除學生

  1. 點選左側的“學生管理”。
  2. 選中您要刪除的學生（支持多選）。
  3. 點選上方的“刪除”按鈕。
  4. 點選“確認”按鈕，確認刪除。

您可能想要：

  • 修改學生
  • 匯入學生
  • 返回到學生管理
  • 返回到頂部

修改學生

  1. 點選左側的“學生管理”。
  2. 選中您要編輯的學生（只能選中單個學生）。
  3. 點選上方的“修改”按鈕。
  4. 按需要修改學生資訊。
  5. 點選“儲存”按鈕。

您可能想要：

  • 匯入學生
  • 刪除學生
  • 返回到學生管理
  • 返回到頂部

匯出學生清單

  1. 點選左側的“學生管理”。
  2. 點選“匯出資料”，將學生帳號匯出為一個 Excel 檔案並下載。

您可能想要：

  • 匯入學生
  • 返回到學生管理
  • 返回到頂部

查找學生

  1. 點選左側的“學生管理”。
  2. 您可以查看系統中已有的學生帳號。點選 ，刷新頁面；點選 ，切換檢視；點選 ，勾選您要顯示的資訊。點選“姓名”或“學籍號”列標題，可以進行排序。
  3. 在“查詢準則”一欄，選定查詢準則。您可以按姓名、學籍號、年級、班級、課程進行搜尋。
  4. 點選“搜尋”按鈕。

您可能想要：

  • 修改學生
  • 返回到學生管理
  • 返回到頂部

課程管理

使用管理員帳號登入後，點選左側的“課程管理”，管理系統中的課程。您可以使用上方的搜尋欄快速找到特定課程。點選 ，可以刷新課程清單；點選 ，切換檢視；點選 ，勾選您要顯示的資訊。您可以對課程做以下操作：

  • 新增課程
  • 修改課程
  • 刪除課程
  • 匯出課程學生清單

新增課程

  1. 點選“課程管理” >> “新增”。
  2. 填寫課程名稱，選擇任課老師。

  3. 點選“儲存”。

注意：任課老師為必填項，因此，在添加課程之前，請先確保您已經匯入了教師帳號。

注意：添加了課程之後，您需要為該課程添加學生，具體請參閱課程匹配。

您可能想要：

  • 為課程添加學生
  • 匯入教師
  • 刪除課程
  • 返回到課程管理
  • 返回到頂部

修改課程

  3. 點選上方的“修改”按鈕。
  4. 按需要修改課程資訊。
  5. 點選“儲存”按鈕。

  3. 在“上課人數”一欄，點選右側的 按鈕。
  4. 該課程的學生清單會匯出為一個 Excel 檔案進行下載。
