Overview

STEMachine is a learning platform for students to practice Python coding and project development. STEMachine delivers an engaging learning experience with the help of a mechanical arm that students can control via Python coding. Through creating and designing projects that manipulate the mechanical arm in different ways, students can discover the fun in coding and project development.

This document will help you learn how to use STEMachine to view student projects and result videos.

Disclaimer:

  • The product description and the images displayed in this document are intended as a guide and the actual product may differ slightly in some respects. Please refer to the actual product.

Login

Open your browser and go to the STEMachine login page:

http://STEMachine address/htm/login.html

Please replace "STEMachine address" with your school's actual STEMachine server address. If you do not know the address, please consult the system administrator or technical support staff.

Note: It is recommended that you use Chrome browser to access STEMachine.

Enter your account and password. Your account is your teacher ID. The default password is 123456. It is recommended that you change your password upon your first login.

You can check Remember password to let the browser remember your password. Click Sign in.

Change Password

  1. On the login page, click Change password.
  2. In the Username field, enter your account (i.e., your teacher ID). Enter your old password (the default password is 123456) and then enter your new password twice.
  3. Click Submit.

Forgot Password

If you have forgotten your password, please contact your system administrator. The system administrator can reset the password for you.

Find a Student

After you have logged in, you will see all your classes listed in the left pane.

Select the class you want to view in the left pane. The students in the selected class will be shown in the right pane. You can search for specific students using the search bar. Click Name or Student ID at the top-right corner to sort the students.

Click a student to view his/her works.

View Student Projects

Click a student to view his/her projects.

In the left pane, you can see a list of the student's projects. You can use the search bar to find specific projects quickly. The status of each project is displayed next to the project name.

Status Description
Unpublished
 The student has uploaded a piece of Python code to the project, but has not published the code yet.
Processing
 A piece of Python code has been uploaded and published by the student. The system is still processing the code and results have not come out yet.
Unwatched
 The results video has been generated, but has not been played yet.
Watched
 The results video has been generated and played.
Error
 A piece of Python code has been uploaded by the student, but the code cannot be executed due to errors.

Click a project to view details.

If the student has already published the code, it will be displayed in the right pane. Click the Results button to view the project result video.

Note: If the code is still being processed and results have not been generated yet, you won't be able to view the video. You can check back later. If there is an error during the execution of the code, you can click the project to view its code. This way, you can learn more about what type of mistakes your students are prone to make and help them improve.

Click Back to class list to go back to the class list.
