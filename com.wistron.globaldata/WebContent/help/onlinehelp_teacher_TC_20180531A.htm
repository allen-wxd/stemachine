簡介

STEMachine 是一款教授學生體驗並使用 Python 語言程式設計的教學平臺。STEMachine 以機械手臂為媒介，讓學生能夠通過基本的 Python 語言應用，操縱機械手臂，體驗專案開發的經驗與樂趣。

本文將為您詳細介紹如何使用 STEMachine 系統查看學生的專案代碼和成果視頻。

聲明：

  • 實際產品中的圖示、文字及實際操作可能與本文檔存在細微差異，請以實際為准。

登入

打開瀏覽器，訪問 STEMachine 登入網址：

http://STEMachine 網址/htm/login.html

其中，請將“STEMachine 網址”替換成您學校實際的 STEMachine 伺服器地址。如果您不知道具體的網址，請諮詢您學校的系統管理員或技術服務人員。

注意：建議您使用 Chrome 瀏覽器訪問 STEMachine。

輸入您的使用者名稱和密碼。使用者名稱是您的教師編號。初始密碼為 123456。建議您在首次登入時立即修改密碼。

如需讓瀏覽器記住您的密碼，可以勾選“記住密碼”選項。點擊“登入”。

修改密碼

  1. 在登入頁面，點擊“修改密碼”。
  2. 在“使用者名稱”一欄，填入您的帳號（即教師編號）。填入舊密碼（初始密碼為 123456），並輸入新密碼兩次。
  3. 點擊“提交”。

忘記密碼

如果您忘記了密碼，請聯繫您的系統管理員。管理員會幫您重置密碼。

您可能想要：

  • 返回到頂部

查找學生

登入後，您將會看到自己教授的班級。

在左側的班級清單中，選擇您要查看的班級。右側欄位元會顯示該班級的學生。您可以直接使用搜尋框查找特定學生。您也可以點擊右側的“姓名”或“學籍號”按鈕，對學生進行排序。

點擊一個學生，查看該學生的作品。

您可能想要：

  • 返回到頂部

查看作品

點擊某個學生，即可查看該學生的作品。

在左側欄位，您可以看到學生的專案清單。您可以使用搜尋框快速查找特定專案。在專案清單中，您可以查看各專案的狀態：

狀態 說明
尚未發佈
 學生已上傳 Python 代碼，但尚未發佈代碼。
等待回傳
 學生已上傳 Python 代碼，並且已發佈代碼。但尚未收到發佈結果。
未播放
 已收到發佈結果，但尚未播放成果視頻。
已播放
 已收到發佈結果，並且已播放成果視頻
結果出錯
 已上傳並發佈 Python 代碼，但代碼中有錯誤，導致代碼無法執行。

點擊某個專案，查看具體內容。

如果學生已經發佈了專案代碼，則右側欄位中會顯示代碼。點擊右上角的“成果”按鈕，可以查看專案的成果視頻。

注意：如果該專案尚未生成結果視頻，則“成果”按鈕為灰色。

點擊“返回班級清單”，返回班級清單頁面。您可以繼續查看其它班級或學生的作品。
