Overview

STEMachine is a learning platform for students to practice Python coding and project development. STEMachine delivers an engaging learning experience with the help of a mechanical arm that students can control via Python coding. Through creating and designing projects that manipulate the mechanical arm in different ways, students can discover the fun in coding and project development.

This document will help you learn how to use the administrator account to import and maintain the student and teacher accounts on the platform.

Disclaimer:

  • The product description and the images displayed in this document are intended as a guide and the actual product may differ slightly in some respects. Please refer to the actual product.

Login

Open your browser and go to the following address:

http://ServerIP:8080/htm/login.html

Please replace "ServerIP" with the IP address of your STEMachine server. Port 8080 is the default port. If you have configured a different port during the system installation, please change "8080" to the port you configured. If you do not know the server address, please consult the technical support person who set up the system for you.

Note: It is recommended that you use Chrome browser to access STEMachine.

Enter the account and password for the administrator account.

Import Students

Before the students can use STEMachine, you need to first import student accounts into the system.

You can add student accounts via the following two ways:

  • Batch Import Student Accounts
  • Create a Student Account Manually

We recommend that you use the Batch Import method to save time. This way, you can add all the student accounts you need in one operation.

Batch Import Student Accounts

  1. Click Manage students >> Import data.
  2. In the Template section, click "学生.xlsx" to download the student account template. The template file is an Excel document. Please make sure you have installed Microsoft Excel before you open it.
  3. Open the template file using Excel. You will see the following table:

  4. In the table, fill in the student ID, grade, class, name, and student number. Since the student ID also serves as the student's login account, please make sure there are no duplicate student IDs in the table. You can refer to the picture below for an example of what a finished student template file looks like. (Please note that the student information in the picture are intended as examples only. Please use your students' real information in your own template file.) After you have finished editing the template file, save it and exit Excel.

  5. On the STEMachine page, go to Manage students >> Import data. Click Choose File and select the template file you made.
  6. Please wait patiently for the file to upload. Afterwards, the file name you selected will appear on the right side of the Choose File button. Click Import data.
  7. After the accounts have been imported, a success message will appear. The accounts you imported will appear on the student list. If you cannot find those accounts, try refreshing the page. If the import failed, a failure message will appear. Please check your template file to make sure it is in the correct format.

Note: The default password for all imported accounts is 123456.

Create a Student Account Manually

  1. Click Manage students >> New.
  2. Fill in the student information.

  3. Click Save.

  • Go back to Manage Students
  • Go back to the top

Assign Courses

After you have added student accounts, you can then assign courses to the students. Before you do that, please make sure you have already imported teacher accounts and added courses.

  1. Click Manage students in the left pane.
  2. Select the students to whom you want to assign courses. Multiple selection is supported.
  3. Click the Assign courses button.
  4. Select the courses you want to assign to the students. Multiple selection is supported.
  5. Click Save.

You may want to:

  • Edit Students
  • Import Students
  • Go back to Manage Students
  • Go back to the top

Delete Students

  1. Click Manage students in the left pane.
  2. Select the students you want to delete (multiple selection is supported).
  3. Click the Delete button.
  4. Click OK to confirm deletion.

You may want to:

  • Edit Students
  • Import Students
  • Go back to Manage Students
  • Go back to the top

Find a Student

  1. Click Manage students in the left pane.
  2. You can view all the student accounts in the system. Click to refresh the list; click to switch view; click to select the detail information you wish to see about each account. Click the Name or Student ID column header to sort by that column.
  3. In the Search criteria section, select the desired search criteria. You can search by name, grade, class and course.
  4. Click Search to perform a search.

You may want to:

  • Edit Students
  • Go back to Manage Students
  • Go back to the top

Manage Courses

Click Manage courses in the left pane to manage courses. You can use the search bar at the top of the right pane to find specific courses quickly. Click to refresh the course list; click to switch view; click to select the detail information you wish to see about each course. You can perform the following actions:

  • Add a Course
  • Edit a Course
  • Delete a Course
  • Export Course Roster

Add a Course

  1. Click Manage courses >> New.
  2. Fill in the course name and select an instructor for the course.

  3. Click Save.

Note: You are required to pick an instructor for the course. So please make sure you have already imported teacher accounts before you start adding courses.

Note: After you have added a course, you need to assign students to the course. Please refer to Assign Courses for details on how to do that.

You may want to:

  • Delete a Course
