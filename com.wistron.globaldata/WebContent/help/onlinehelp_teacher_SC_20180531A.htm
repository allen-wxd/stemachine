简介

STEMachine 是一款教授学生体验并使用 Python 语言编程的教学平台。STEMachine 以机械手臂为媒介，让学生能够通过基本的 Python 语言应用，操纵机械手臂，体验项目开发的经验与乐趣。

本文将为您详细介绍如何使用 STEMachine 系统查看学生的项目代码和成果视频。

声明：

  • 实际产品中的图示、文字及实际操作可能与本文档存在细微差异，请以实际为准。

登录

打开浏览器，访问 STEMachine 登录网址：

http://STEMachine 网址/htm/login.html

其中，请将“STEMachine 网址”替换成您学校实际的 STEMachine 服务器地址。如果您不知道具体的网址，请咨询您学校的系统管理员或技术服务人员。

注意：建议您使用 Chrome 浏览器访问 STEMachine。

输入您的用户名和密码。用户名是您的教师编号。初始密码为 123456。建议您在首次登录时立即修改密码。

如需让浏览器记住您的密码，可以勾选“记住密码”选项。点击“登录”。

修改密码

  1. 在登录页面，点击“修改密码”。
  2. 在“用户名”一栏，填入您的账号（即教师编号）。填入旧密码（初始密码为 123456），并输入新密码两次。
  3. 点击“提交”。

忘记密码

如果您忘记了密码，请联系您的系统管理员。管理员会帮您重置密码。

您可能想要：

  • 返回到顶部

查找学生

登录后，您将会看到自己教授的班级。

在左侧的班级列表中，选择您要查看的班级。右侧栏位会显示该班级的学生。您可以直接使用搜索框查找特定学生。您也可以点击右侧的“姓名”或“学籍号”按钮，对学生进行排序。

点击一个学生，查看该学生的作品。

您可能想要：

  • 返回到顶部

查看作品

点击某个学生，即可查看该学生的作品。

在左侧栏位，您可以看到学生的项目列表。您可以使用搜索框快速查找特定项目。在项目列表中，您可以查看各项目的状态：

状态 说明
尚未发布
 学生已上传 Python 代码，但尚未发布代码。
等待回传
 学生已上传 Python 代码，并且已发布代码。但尚未收到发布结果。
未播放
 已收到发布结果，但尚未播放成果视频。
已播放
 已收到发布结果，并且已播放成果视频
结果出错
 已上传并发布 Python 代码，但代码中有错误，导致代码无法执行。

点击某个项目，查看具体内容。

如果学生已经发布了项目代码，则右侧栏位中会显示代码。点击右上角的“成果”按钮，可以查看项目的成果视频。

注意：如果该项目尚未生成结果视频，则“成果”按钮为灰色。

点击“返回班级列表”，返回班级列表页面。您可以继续查看其它班级或学生的作品。
