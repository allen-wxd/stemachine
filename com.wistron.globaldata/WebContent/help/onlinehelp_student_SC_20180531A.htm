简介

STEMachine 是一款教授学生体验并使用 Python 语言编程的教学平台。STEMachine 以机械手臂为媒介，让学生能够通过基本的 Python 语言应用，操纵机械手臂，体验项目开发的经验与乐趣。

本文将为您详细介绍如何使用 STEMachine 系统，自主完成项目创建和代码编辑，并最终通过代码成功操纵机械手臂，您还可以通过视频来观看和展示自己的开发成果哦！

您可以通过以下流程，轻松完成项目开发的各项步骤，体验创作的乐趣。点击一个步骤，查看详细内容。

创建项目 上传代码 语法检查 代码发布 成果展示

  • 实际产品中的图示、文字及实际操作可能与本文档存在细微差异，请以实际为准。

登录

打开浏览器，访问 STEMachine 登录网址：

http://STEMachine 网址/htm/login.html

其中，请将“STEMachine 网址”替换成您学校实际的 STEMachine 服务器地址。如果您不知道具体的网址，请咨询您的老师或技术服务人员。

注意：建议您使用 Chrome 浏览器访问 STEMachine。

输入您的用户名和密码。用户名是您的学籍号。初始密码为 123456。建议您在首次登录时立即修改密码。

如需让浏览器记住您的密码，可以勾选“记住密码”选项。点击“登录”。

修改密码

  1. 在登录页面，点击“修改密码”。
  2. 在“用户名”一栏，填入您的账号（即学籍号）。填入旧密码（初始密码为 123456），并输入新密码两次。
  3. 点击“提交”。

忘记密码

如果您忘记了密码，请联系您的任课老师。老师会帮您重置密码。

您可能想要：

  • 返回到顶部

项目管理

在上传代码之前，您首先需要创建一个项目。您可以创建多个项目。每个项目都可以单独进行视频生成和成果展示。

点击屏幕上方的“代码管理”和“成果展示”，可以在代码视图和成果展示之间切换。在左侧栏位中，您可以看到自己的项目列表，以及各项目的状态：

状态 说明
尚未发布
 已上传 Python 代码，但尚未发布代码。
等待回传
 已上传 Python 代码，并且已发布代码。但尚未收到发布结果。
未播放
 已收到发布结果，但尚未播放成果视频。
已播放
 已收到发布结果，并且已播放成果视频
结果出错
 已上传并发布 Python 代码，但代码中有错误，导致代码无法执行。

在左侧栏位中，您也可以使用上方的搜索栏，快速找到特定项目，本章节将为您介绍以下内容：

  • 新建项目
  • 重命名项目
  • 删除项目

您可能想要：

  • 返回到顶部

新建项目

  1. 进入“代码管理”视图。
  2. 在右侧栏位中，点击“新建项目”。
  3. 输入项目名称，点击“保存”。

您可能想要：

  • 上传代码
  • 删除项目
  • 返回到项目管理
  • 返回到顶部

重命名项目

  1. 进入“代码管理”视图。
  2. 在左侧栏位中，选中您要重命名的项目。
  3. 在右侧栏位中，点击上方的“编辑项目”。
  4. 输入新的项目名称。
  5. 点击“保存”按钮。

您可能想要：

  • 返回到项目管理
  • 返回到顶部

删除项目

  1. 进入“代码管理”视图。
  2. 在左侧栏位中，选中您要删除的项目。
  3. 在右侧栏位中，点击“删除项目”。
  4. 点击“确认”，确认删除。

您可能想要：

  • 返回到项目管理
  • 返回到顶部

代码管理

在创建了项目以后，您就可以为项目上传和发布代码了。

本章节将为您介绍以下内容：

  • 上传代码
  • 语法检查
  • 代码发布

您可能想要：

  • 返回到顶部

上传代码

  1. 进入“代码管理”视图。
  2. 在左侧栏位中，选中您要上传代码的项目。
  3. 在右侧栏位中，请点击“上传代码”。
  4. 点击“选择文件”，选择您写好的 Python 代码文件。您可以上传 .txt 或 .py 格式的文件。
  5. 请耐心等待文件上传。上传完毕后，在“选择文件”按钮右侧，会出现您所选文件的文件名。点击“上传”，即可导入代码。

您可能想要：

  • 新建项目
  • 返回到代码管理
  • 返回到顶部

语法检查

  1. 上传代码后，点击“语法检查”。
  2. 如果发现错误，页面上会弹出窗口，显示具体的错误行数及内容。如果没有发现错误，则不会弹出提示，并且右侧的“代码发布”按钮会变成黄色。表示代码已经可以发布了。
  3. 如果语法检查发现了错误，您可以直接在代码框内对代码进行修改。修改完毕后，请再次点击“语法检查”。当没有错误以后，就可以点击“代码发布”并保存修改了。如果您在尚未发布代码之前就跳转到其他页面，系统会询问您是否要临时保存代码，点击“确认”，保存您对代码作的修改。您可以下一次再继续编辑。

您可能想要：

  • 上传代码
  • 返回到代码管理
  • 返回到顶部

代码发布

  1. 语法检查后，当没有发现错误时，“代码发布”按钮会从灰色转为黄色，点击该按钮，再点击“确认”即可发布代码。

注意：只有通过了语法检查，才能进行代码发布。

您可能想要：

  • 成果展示
  • 上传代码
  • 返回到代码管理
  • 返回到顶部

成果展示

发布了代码以后，系统就会执行代码来操作机械手臂，并用视频记录手臂的运动情形。

如果项目代码执行成功，并且视频已经生成，则项目右下角的状态标签会显示“未播放”或“已播放”。这时，您就可以播放成果视频了。请在左侧栏位中选中要播放的项目，点击页面上方的“成果展示”，进入视频播放页面。

下载视频

点击视频右下角的 ，可以下载视频到本地。

如果您发布了某个项目的代码，但是无法查看项目视频，这可能是由于：

  • 发布结果尚未生成，您可以稍后再查看
  • 代码在执行过程中出现了问题

这时，您可以切换到“代码管理”视图。在左侧的栏位中，您可以看到项目列表和状态：

状态 说明
尚未发布
 已上传 Python 代码，但尚未发布代码。
等待回传
 已上传 Python 代码，并且已发布代码。但尚未收到发布结果。请耐心等待结果生成，您可以稍后再来查看。
未播放
 已收到发布结果，但尚未播放成果视频。
已播放
 已收到发布结果，并且已播放成果视频
结果出错
 已上传并发布 Python 代码，但代码中有错误，导致代码无法执行。

如果状态为“结果出错”，您可以修改出错的代码，然后重新进行语法检查和代码发布。

您可能想要：

  • 返回到成果展示
  • 返回到顶部
