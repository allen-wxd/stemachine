簡介

STEMachine 是一款教授學生體驗並使用 Python 語言程式設計的教學平臺。STEMachine 以機械手臂為媒介，讓學生能夠通過基本的 Python 語言應用，操縱機械手臂，體驗專案開發的經驗與樂趣。

本文將為您詳細介紹如何使用 STEMachine 系統，自主完成專案創建和代碼編輯，並最終通過代碼成功操縱機械手臂，您還可以通過視頻來觀看和展示自己的開發成果哦！

您可以通過以下流程，輕鬆完成專案開發的各項步驟，體驗創作的樂趣。點擊一個步驟，查看詳細內容。

創建專案 上傳代碼 語法檢查 代碼發佈 成果展示

聲明：

  • 實際產品中的圖示、文字及實際操作可能與本文檔存在細微差異，請以實際為准。

登入

打開瀏覽器，訪問 STEMachine 登入網址：

http://STEMachine 網址/htm/login.html

其中，請將“STEMachine 網址”替換成您學校實際的 STEMachine 伺服器地址。如果您不知道具體的網址，請諮詢您的老師或技術服務人員。

注意：建議您使用 Chrome 瀏覽器訪問 STEMachine。

輸入您的使用者名稱和密碼。使用者名稱是您的學籍號。初始密碼為 123456。建議您在首次登入時立即修改密碼。

如需讓瀏覽器記住您的密碼，可以勾選“記住密碼”選項。點擊“登入”。

修改密碼

  1. 在登入頁面，點擊“修改密碼”。
  2. 在“使用者名稱”一欄，填入您的帳號（即學籍號）。填入舊密碼（初始密碼為 123456），並輸入新密碼兩次。
  3. 點擊“提交”。

忘記密碼

如果您忘記了密碼，請聯繫您的任課老師。老師會幫您重置密碼。

您可能想要：

  • 返回到頂部

專案管理

在上傳代碼之前，您首先需要創建一個專案。您可以創建多個專案。每個專案都可以單獨進行視頻生成和成果展示。

點擊螢幕上方的“代碼管理”和“成果展示”，可以在代碼檢視和成果展示之間切換。在左側欄位中，您可以看到自己的專案清單，以及各專案的狀態：

狀態 說明
尚未發佈
 已上傳 Python 代碼，但尚未發佈代碼。
等待回傳
 已上傳 Python 代碼，並且已發佈代碼。但尚未收到發佈結果。
未播放
 已收到發佈結果，但尚未播放成果視頻。
已播放
 已收到發佈結果，並且已播放成果視頻
結果出錯
 已上傳並發佈 Python 代碼，但代碼中有錯誤，導致代碼無法執行。

在左側欄位中，您也可以使用上方的搜尋欄，快速找到特定專案，本章節將為您介紹以下內容：

  • 新增專案
  • 重新命名專案
  • 刪除專案

您可能想要：

  • 返回到頂部

新增專案

  1. 進入“代碼管理”檢視。
  2. 在右側欄位中，點擊“新增專案”。
  3. 輸入專案名稱，點擊“儲存”。

您可能想要：

  • 上傳代碼
  • 刪除專案
  • 返回到專案管理
  • 返回到頂部

重新命名專案

  1. 進入“代碼管理”檢視。
  2. 在左側欄位中，選中您要重新命名的專案。
  3. 在右側欄位中，點擊上方的“編輯專案”。
  4. 輸入新的專案名稱。
  5. 點擊“儲存”按鈕。

您可能想要：

  • 返回到專案管理
  • 返回到頂部

刪除專案

  1. 進入“代碼管理”檢視。
  2. 在左側欄位中，選中您要刪除的專案。
  3. 在右側欄位中，點擊“刪除專案”。
  4. 點擊“確認”，確認刪除。

您可能想要：

  • 返回到專案管理
  • 返回到頂部

代碼管理

在創建了專案以後，您就可以為專案上傳和發佈代碼了。

本章節將為您介紹以下內容：

  • 上傳代碼
  • 語法檢查
  • 代碼發佈

您可能想要：

  • 返回到頂部

上傳代碼

  1. 進入“代碼管理”檢視。
  2. 在左側欄位中，選中您要上傳代碼的專案。
  3. 在右側欄位中，請點擊“上傳代碼”。
  4. 點擊“選擇檔案”，選擇您寫好的 Python 代碼檔案。您可以上傳 .txt 或 .py 格式的檔案。
  5. 請耐心等待文件上傳。上傳完畢後，在“選擇檔案”按鈕右側，會出現您所選檔案的檔案名。點擊“上傳”，即可匯入代碼。

您可能想要：

  • 新增專案
  • 返回到代碼管理
  • 返回到頂部

語法檢查

  1. 上傳代碼後，點擊“語法檢查”。
  2. 如果發現錯誤，頁面上會快顯視窗，顯示具體的錯誤行數及內容。如果沒有發現錯誤，則不會彈出提示，並且右側的“代碼發佈”按鈕會變成黃色。表示代碼已經可以發佈了。
  3. 如果語法檢查發現了錯誤，您可以直接在代碼框內對代碼進行修改。修改完畢後，請再次點擊“語法檢查”。當沒有錯誤以後，就可以點擊“代碼發佈”並儲存修改了。如果您在尚未發佈代碼之前就跳轉到其他頁面，系統會詢問您是否要臨時儲存代碼，點擊“確認”，儲存您對代碼作的修改。您可以下一次再繼續編輯。

您可能想要：

  • 上傳代碼
  • 返回到代碼管理
  • 返回到頂部

代碼發佈

  1. 語法檢查後，當沒有發現錯誤時，“代碼發佈”按鈕會從灰色轉為黃色，點擊該按鈕，再點擊“確認”即可發佈代碼。

注意：只有通過了語法檢查，才能進行代碼發佈。

您可能想要：

  • 成果展示
  • 上傳代碼
  • 返回到代碼管理
  • 返回到頂部

成果展示

發佈了代碼以後，系統就會執行代碼來操作機械手臂，並用視頻記錄手臂的運動情形。

如果專案代碼執行成功，並且視頻已經生成，則專案右下角的狀態標籤會顯示“未播放”或“已播放”。這時，您就可以播放成果視頻了。請在左側欄位中選中要播放的專案，點擊頁面上方的“成果展示”，進入視頻播放頁面。

下載視頻

點擊視頻右下角的 ，可以下載視頻到本地。

如果您發佈了某個專案的代碼，但是無法查看專案視頻，這可能是由於：

  • 發佈結果尚未生成，您可以稍後再查看
  • 代碼在執行過程中出現了問題

這時，您可以切換到“代碼管理”檢視。在左側的欄位中，您可以看到專案清單和狀態：

狀態 說明
尚未發佈
 已上傳 Python 代碼，但尚未發佈代碼。
等待回傳
 已上傳 Python 代碼，並且已發佈代碼。但尚未收到發佈結果。請耐心等待結果生成，您可以稍後再來查看。
未播放
 已收到發佈結果，但尚未播放成果視頻。
已播放
 已收到發佈結果，並且已播放成果視頻
結果出錯
 已上傳並發佈 Python 代碼，但代碼中有錯誤，導致代碼無法執行。

如果狀態為“結果出錯”，您可以修改出錯的代碼，然後重新進行語法檢查和代碼發佈。

您可能想要：

  • 返回到成果展示
  • 返回到頂部
