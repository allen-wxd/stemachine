简介

STEMachine 是一款教授学生体验并使用 Python 语言编程的教学平台。STEMachine 以机械手臂为媒介，让学生能够通过基本的 Python 语言应用，操纵机械手臂，体验项目开发的经验与乐趣。

本文将为您详细介绍如何使用管理员账号导入和维护系统上的学生和教师账号。

声明：

  • 实际产品中的图示、文字及实际操作可能与本文档存在细微差异，请以实际为准。

登录

打开浏览器，访问 STEMachine 网址：

http://ServerIP:8080/htm/login.html

其中，请将 ServerIP 替换成您实际的 STEMachine 服务器 IP 地址。8080 为默认端口，如果您在系统配置时设置了其他端口，请替换为其他端口号。如果您不知道服务器 IP 地址，请咨询帮您搭建 STEMachine 系统的技术服务人员。

注意：建议您使用 Chrome 浏览器访问 STEMachine。

输入管理员账号的用户名和密码：

用户名：admin

初始密码：123456

点击“登录”。

您可能想要：

  • 返回到顶部

学生管理

使用管理员账号登录后，点击左侧的“学生管理”，管理学生账号。您可以对学生账号做以下操作：

  • 导入学生
  • 课程匹配
  • 删除学生
  • 修改学生
  • 导出学生列表
  • 查找学生

导入学生

在学生可以使用 STEMachine 系统之前，您首先需要导入学生账号。

有以下两种方式可以添加学生账号：

  • 批量导入学生
  • 手动添加学生

建议您使用批量导入的方式，以节省时间。您只需一次操作，就可以添加所有学生账号。

批量导入学生

  1. 点击“学生管理” >> “导入数据”。
  2. 在“参考模板”一栏，点击“学生.xlsx”，下载学生账号模板文件。模板文件为 Excel 格式，使用前请确保您已经安装了 Microsoft Excel。
  3. 用 Excel 打开您下载的模板文件。您将会看到诸如下图格式的表格：

  4. 在表格中填入每个学生的学籍号、年级、班级、姓名和学号。姓名长度需在 2-8 个字符范围内。其中，学籍号即为学生的登录账号，因此，每个学生的学籍号不得重复。如下图所示（图中文字仅为举例说明用，实际操作中请填入真实的学生信息）。填写完毕后，请保存文件并退出 Excel。

  5. 在 STEMachine 页面，前往“学生管理” >> “导入数据”。点击“选择文件”，选择您刚才制作好的 Excel 文件。
  6. 请耐心等待文件上传。上传完毕后，在“选择文件”按钮右侧，会出现您所选文件的文件名。完成后，点击“导入数据”。
  7. 导入成功后，会出现提示，您可以在学生列表中看到导入的学生账号。如果没有看到账号，可以尝试刷新页面。如果导入失败，屏幕会出现失败提示消息。请对比模板，检查您的 Excel 文件格式是否正确。

注意：导入账号的密码将自动设为默认密码“123456”。

手动添加学生

  1. 点击“学生管理” >> “新增”。
  2. 填写学生信息。

  3. 点击“保存”。

您可能想要：

  • 导入教师
  • 添加课程
  • 修改学生
  • 删除学生
  • 返回到学生管理
  • 返回到顶部

课程匹配

添加了学生账号之后，您还可以为学生分配课程。在分配课程之前，请确保您已经导入了教师账号并添加了相应的课程。

  1. 点击左侧的“学生管理”。
  2. 选中您要分配课程的学生（支持多选）。
  3. 点击上方的“课程匹配”按钮。
  4. 勾选您要给这些学生分配的课程（支持多选）。
  5. 点击“保存”按钮。

您可能想要：

  • 修改学生
  • 导入学生
  • 返回到学生管理
  • 返回到顶部

删除学生

  1. 点击左侧的“学生管理”。
  2. 选中您要删除的学生（支持多选）。
  3. 点击上方的“删除”按钮。
  4. 点击“确认”按钮，确认删除。

您可能想要：

  • 修改学生
  • 导入学生
  • 返回到学生管理
  • 返回到顶部

修改学生

  1. 点击左侧的“学生管理”。
  2. 选中您要编辑的学生（只能选中单个学生）。
  3. 点击上方的“修改”按钮。
  4. 按需要修改学生信息。
  5. 点击“保存”按钮。

您可能想要：

  • 导入学生
  • 删除学生
  • 返回到学生管理
  • 返回到顶部

导出学生列表

  1. 点击左侧的“学生管理”。
  2. 点击“导出数据”，将学生账号导出为一个 Excel 文件并下载。

您可能想要：

  • 导入学生
  • 返回到学生管理
  • 返回到顶部

查找学生

  1. 点击左侧的“学生管理”。
  2. 您可以查看系统中已有的学生账号。点击 ，刷新列表；点击 ，切换视图；点击 ，勾选您要显示的信息。点击“姓名”或“学籍号”列标题，可以进行排序。
  3. 在“查询条件”一栏，选定查询条件。您可以按姓名、学籍号、年级、班级、课程进行搜索。
  4. 点击“搜索”按钮。

您可能想要：

  • 修改学生
  • 返回到学生管理
  • 返回到顶部

课程管理

使用管理员账号登录后，点击左侧的“课程管理”，管理系统中的课程。您可以使用上方的搜索栏快速找到特定课程。点击 ，可以刷新课程列表；点击 ，切换视图；点击 ，勾选您要显示的信息。您可以对课程做以下操作：

  • 新增课程
  • 修改课程
  • 删除课程
  • 导出课程学生列表

新增课程

  1. 点击“课程管理” >> “新增”。
  2. 填写课程名称，选择任课老师。

  3. 点击“保存”。

注意：任课老师为必填项，因此，在添加课程之前，请先确保您已经导入了教师账号。

注意：添加了课程之后，您需要为该课程添加学生，具体请参阅课程匹配。

您可能想要：

  • 为课程添加学生
  • 导入教师
  • 删除课程
  • 返回到课程管理
  • 返回到顶部
